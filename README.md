# Iris Species Classification - Streamlit App

This repository contains a Streamlit app that trains a Random Forest classifier on the Iris dataset,
shows model evaluation metrics, and lets users input measurements to predict the species.


## Files
- `Proyect.py`: Main Streamlit application.
- `requirements.txt`: Python dependencies.
